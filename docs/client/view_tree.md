클라이언트 화면
├─ 첫 화면
│  ├─ 게임 소개 구역
│  │  ├─ 게임 이름
│  │  ├─ 한 줄 설명
│  │  └─ 어떻게 시작하는지 설명
│  ├─ 닉네임 입력 구역
│  │  ├─ 닉네임 입력칸
│  │  ├─ 입력 안내 문구
│  │  └─ 입력 오류 문구
│  ├─ 시작 버튼 구역
│  │  ├─ 무작위 매칭 버튼
│  │  └─ 방 만들기 버튼
│  └─ 입장 실패 안내 구역
│     ├─ 오류 문구
│     └─ 다시 시도 안내
│
└─ 게임 화면
   ├─ 맨 위 상태 표시줄 (항상 보임)
   │  ├─ 몇 라운드인지
   │  ├─ 남은 시간
   │  └─ 이번 라운드 그림 순서
   │     (타원형 칸들로 한 줄 표시. 줄바꿈 없음. 좌우 스크롤 가능.
   │      턴이 지날 때마다 현재 턴보다 앞선 사람 칸은 하나씩 사라짐)
   │
   ├─ 본문 3칸 영역
   │  ├─ 왼쪽 구역 (높이는 가운데 구역과 같음)
   │  │  ├─ 방 코드 구역
   │  │  │  ├─ 방 코드 표시
   │  │  │  └─ 링크 복사 버튼
   │  │  └─ 참여자 정보 구역
   │  │     ├─ 참여자 목록
   │  │     │  └─ 참여자 한 칸
   │  │     │     ├─ 닉네임
   │  │     │     ├─ 점수
   │  │     │     ├─ 방장 표시 (Host 딱지)
   │  │     │     ├─ 현재 그림 그리는 사람 표시 (굵은 테두리)
   │  │     │     ├─ 정답 맞춘 사람 표시 (칸 색을 초록색으로 표시)
   │  │     │     └─ 잠깐 뜨는 말풍선 (칸 오른쪽에 잠깐 떴다가 3초 후 사라짐)
   │  │     └─ 목록이 길어지면 내부 스크롤
   │  │
   │  ├─ 가운데 구역
   │  │  ├─ 그림판 바깥 틀
   │  │  │  ├─ 그림이 실제로 그려지는 곳
   │  │  │  │  (게임 시작 전에는 모두 그림 가능.
   │  │  │  │   설정 화면이 열려 있으면 그림 불가.
   │  │  │  │   게임 진행 중에는 drawer만 그림 가능)
   │  │  │  ├─ 그림 못 그리는 사람에게 보여주는 덮개 문구
   │  │  │  ├─ 정답 단어 표시 구역
   │  │  │  │  (drawer에게만 보일 수 있음)
   │  │  │  ├─ 방장 전용 설정 열기 버튼
   │  │  │  │  (그림 영역 오른쪽 위. 방장에게만 보임)
   │  │  │  └─ 그림판 위에 뜨는 상태 화면
   │  │  │     ├─ 게임 시작 전 설정 화면
   │  │  │     │  (그림판 위에 뜸. 열기/닫기 가능.
   │  │  │     │   방장만 설정 변경 가능. 시작 버튼 포함.
   │  │  │     │   열려 있는 동안에는 그림 불가. 채팅은 가능)
   │  │  │     ├─ 단어 고르기 화면
   │  │  │     ├─ 라운드 시작 안내 화면
   │  │  │     │  (이벤트가 오면 잠깐 보여줌)
   │  │  │     ├─ 턴 진행 안내 화면
   │  │  │     │  (지금 누구 차례인지, 다음은 누구인지 포함)
   │  │  │     ├─ 턴 끝난 결과 화면
   │  │  │     │  (정답 맞춘 사람, 획득 점수 표시.
   │  │  │     │   획득 점수 높은 순으로 표처럼 보여줌)
   │  │  │     ├─ 라운드 넘어가는 안내 화면
   │  │  │     │  (없어도 됨. 바로 다음 라운드 시작 화면으로 넘어가도 됨)
   │  │  │     └─ 게임 결과 화면
   │  │  │        (그림판 위에 반투명하게 뜸.
   │  │  │         지난 그림은 뒤에 비치듯 보임.
   │  │  │         1, 2, 3등만 표시.
   │  │  │         세로 중앙 정렬.
   │  │  │         1등은 크게 초록색 글씨, 나머지는 작고 검은 글씨)
   │  │  └─ 그림 도구 구역
   │  │     (그림판 바깥 틀 바로 아래. 항상 보임)
   │  │     ├─ 펜
   │  │     ├─ 지우개
   │  │     ├─ 전체 지우기
   │  │     ├─ 굵기 조절
   │  │     └─ 색상 선택
   │  │
   │  └─ 오른쪽 구역 (높이는 가운데 구역과 같음)
   │     └─ 채팅 구역
   │        ├─ 채팅 목록
   │        │  ├─ 시스템 안내 글 (파란 글씨, 굵게)
   │        │  ├─ 일반 추측 글 (검은 글씨)
   │        │  ├─ 정답 맞춘 사람 글 (초록 글씨)
   │        │  └─ 일부 사람에게만 보이는 글 (주황 글씨)
   │        │  (최하단을 보고 있을 때 새 채팅이 들어오면 자동 스크롤.
   │        │   목록이 길어지면 내부 스크롤)
   │        └─ 채팅 입력 구역
   │           ├─ 입력칸
   │           ├─ 전송 버튼
   │           └─ 입력 막힘 상태
   │           (항상 보임. 모든 상태에서 입력 가능)
   │
   └─ 화면 동작 메모
      ├─ 게임 시작 전에는
      │  ├─ 모두 그림 가능
      │  ├─ 모두 채팅 가능
      │  ├─ 방장만 설정 버튼 사용 가능
      │  └─ 설정 화면이 열려 있으면 그림 불가
      ├─ 게임 진행 중에는
      │  ├─ drawer만 그림 가능
      │  ├─ 채팅은 계속 가능
      │  └─ 가운데 상태 화면만 이벤트에 따라 바뀜
      └─ 게임 종료 후에는
         ├─ 마지막 그림이 뒤에 남아 있음
         └─ 그 위에 결과 화면이 뜸
